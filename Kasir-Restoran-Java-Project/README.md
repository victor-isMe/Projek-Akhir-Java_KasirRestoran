# Aplikasi Kasir Restoran Dapur Nusantara

Aplikasi Kasir Restoran **Dapur Nusantara** merupakan program berbasis *console* yang dikembangkan menggunakan bahasa pemrograman **Java**. Aplikasi ini dibuat untuk membantu proses pengelolaan menu makanan dan minuman, pencatatan pesanan pelanggan, perhitungan total harga, serta pencatatan riwayat transaksi secara sederhana dan terstruktur.

## Fitur Aplikasi
Aplikasi ini memiliki beberapa fitur utama, yaitu:
1. Menambahkan menu makanan dan minuman
2. Menampilkan daftar menu beserta harga
3. Mengubah dan menghapus menu makanan dan minuman
4. Mencatat pesanan pelanggan
5. Mengecek status pesanan berdasarkan ID Pesanan
6. Menghitung total harga pesanan dan proses pembayaran
7. Menampilkan riwayat seluruh pesanan
8. Keluar dari program

## Konsep yang Digunakan
Dalam pengembangan aplikasi ini, diterapkan beberapa konsep dasar pemrograman, antara lain:
- Penggunaan class
- Penggunaan struktur data dinamis `ArrayList`
- Percabangan (`switch-case`, `if-else`, `nested if`)
- Perulangan (`while`, `do-while`, `for`)
- Input dan output menggunakan `Scanner`

## Struktur Program
Struktur folder pada repository ini adalah sebagai berikut:

`Kasir-Restoran-Java-Project/`
`├── KasirRestoran.java`
`├── README.md`

## Cara Menjalankan Program
1. Pastikan **Java Development Kit (JDK)** sudah terpasang di komputer
2. Buka terminal atau command prompt
3. Masuk ke folder `Kasir-Restoran-Java-Project`
4. Compile program dengan perintah:
    `javac KasirRestoran.java`
5. Jalankan program dengan perintah:
    `java KasirRestoran`
